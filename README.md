# 🚀 Task App – Fullstack SaaS

Aplicación moderna de gestión de tareas con autenticación, drag & drop y sistema multiusuario.

---

## 🌐 Live Demo

👉 Frontend: https://task-frontend-ten-sigma.vercel.app  
👉 Backend: https://task-api-emanuel.onrender.com 

---

## ✨ Features

- 🔐 Autenticación con Supabase
- 📝 CRUD de tareas
- 🎯 Drag & Drop persistente
- 👥 Compartir tareas entre usuarios
- 🧑‍💼 Roles básicos (admin / user)
- ⚡ Animaciones (Framer Motion)
- 🔔 Notificaciones (toast)
- 🌍 Deploy (Vercel + Render)

---

## 🧠 Arquitectura

**Frontend**
- React (Vite)
- Tailwind CSS
- Framer Motion
- @hello-pangea/dnd

**Backend**
- Node.js + Express
- Prisma ORM
- SQLite

**Auth**
- Supabase

---
